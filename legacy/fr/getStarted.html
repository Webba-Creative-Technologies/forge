WSS :

Qu'est-ce que c'est ?

WSS (Webba Style System) est une librairie web développée par Webba qui vous aidera à devenir plus productif et à générer des sites web beaucoup plus rapidement.

Recommandé

CSS

Copiez-collez ce code dans votre head avant vos propres feuilles de style :

<"link href="https://software.webba-creative.com/wss/wss.css" rel="stylesheet">

JS

Copiez-collez ce code à la fin de votre body :

<script src="https://software.webba-creative.com/wss/js/wss.js"></script>
